      What is a program?

      A program is a specification for the behavior of a program. “It tells the computer what to do”

      The action or response might be simple or very complex, however it consists of simple instructions that combine to create the complex responses .

      (which is why it's important to know what these simple instructions are)

      The programmer’s job is to write what the computer does and what happens when the user interacts with the program.

      Examples:

      • The program is started
      • The user clicks a button
      • The user presses enter
      • A message arrives over the network
      • A camera is turned on

      Instructions

      What do programs consist of?

      • Written of instructions for what the program should do
      • Descriptions of response to events/requests/functionalities that the program needs a behavior for
      • A language that the computer can understand (any programming language, Eg: Java)

      Examples of instructions:

      • Store information such as numbers or text in variables.
      • Computers are calculators so they can do math such as
        • Store this number in a variable (x = 10 or y = 41)
        • Doing math using variables (z = x + y - 4)
      • Store “a string” of text as a variable (text = “This is a string of text”)
      • Merge several strings of text (newtext = text + “” + “and it is now longer!”)
      • Do something depending on a condition
      • Tell the computer to repeat something many times
